# NE555 Generator

## Application:
NE555 circuit can operate like oscilator known as astable multivibrator.
The circuit can be used to continously switch output state. For example generated wave can falsh lights or produce sound.

## Specification: 
- *Vcc - 9 [V]*
- *f - 2 [Hz]*
- *T - 0,5 [s]*
- *pulse width - 50%*
